Entendendo os principios da DOM

DOM (Document Object Model) é uma representação em forma de árvore de objetos do conteudo do HTML.

Para que serve?

Serve para controlar e modificar dinamicamente o conteúdo, estrutura, estilo e etc usando JS.

Com o DOM é possível:

  • Alterar texto de elementos
  • Aplicar ou remover CSS
  • Reagir a eventos(cliques, teclados etc)
  • Trocar imagens
  • Adicicionar/remover elementos
  • Criar animações simples
